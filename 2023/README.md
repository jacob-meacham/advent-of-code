[2023 Advent of Code](https://adventofcode.com/2023) with C#!

# Usage

* Build with `build.sh`
* To run a day, use `cd Day<x> && dotnet run Day<x>.csproj`
* To run benchmarks, use `python3 runner.py --table-only`
